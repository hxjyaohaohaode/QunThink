# 群聊核心问题修复与功能增强 — 验收检查清单

## 核心Bug修复验证

- [x] 用户发送一条消息后，AI不再说"你发了好几遍"或"你重复发了"
- [x] AI收到的上下文中用户消息只出现一次（通过日志或调试确认）
- [x] buildAPIMessages中recentMessages不包含与userMessage重复的内容
- [x] AI不直接复制粘贴其他AI的原话作为自己的回复
- [x] AI在同意其他AI观点时用自己的风格重新表达

## AI时间感知验证

- [x] AI的system prompt中包含当前日期和时间
- [x] AI能自然地提及当前时间（如"今天"、"现在"、"这么晚了"等）
- [x] 时间信息在每次AI调用时都是最新的

## 消息引用功能验证

- [x] AI回复特定消息时，消息包含reply_to字段
- [x] 前端正确显示引用卡片（发送者+内容摘要）
- [x] 用户点击回复引用按钮后，输入框显示引用预览
- [x] 用户发送引用回复后，消息关联被引用消息
- [x] AI收到引用上下文后知道用户在回复哪条消息

## 停止对话功能验证

- [x] AI正在回复时，停止按钮可见
- [x] 没有AI在回复时，停止按钮隐藏
- [x] 点击停止按钮后，所有AI立即停止生成
- [x] 停止后typing状态正确清除
- [x] 停止后已生成的部分消息不保存到数据库
- [x] WebSocket正确传递stop_generation和generation_stopped消息

## 消息操作面板验证

- [x] 每条消息hover时显示操作面板
- [x] 操作面板包含：点赞👍、点踩👎、评论💬、回复引用↩️
- [x] 点赞按钮点击后消息点赞数+1，按钮高亮
- [x] 再次点击点赞取消，点赞数-1
- [x] 点踩按钮点击后消息点踩数+1，按钮高亮
- [x] 再次点击点踩取消，点踩数-1
- [x] 评论按钮点击后展开评论区域
- [x] 用户可以输入评论并提交
- [x] 评论显示发送者头像和名称
- [x] 回复引用按钮点击后设置引用状态，输入框显示预览
- [x] 消息数据模型包含dislikes和disliked_by字段

## AI反馈验证

- [x] AI消息被点赞后，AI在后续回复中可能自然提及反馈
- [x] AI消息被点踩后，AI在后续回复中可能调整观点或表达不满
- [x] 反馈信息通过system prompt注入，不强制AI回应

## 私聊模式验证

- [ ] 侧边栏AI成员列表可点击进入私聊
- [ ] 私聊界面与群聊风格一致，只有用户和单个AI
- [ ] 私聊消息正确发送和接收
- [ ] 私聊历史记录持久化，重新打开可看到
- [ ] 私聊记忆在群聊中作为额外上下文注入
- [ ] AI在群聊中可参考私聊记忆（自行决定是否使用）
- [ ] 私聊和群聊可平滑切换
- [ ] WebSocket正确处理私聊频道
