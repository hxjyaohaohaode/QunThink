# 群聊自然交互与未读消息体验优化 — 任务清单

## 第一批：后端 — AI自然发言调度重构

- [x] Task 1: 重构发言调度引擎，移除顺序延迟
  - [x] 1.1: 修改 `calculateInitialDelay`，移除 `index * 500` 顺序因子，改为完全独立的随机延迟（1~8秒范围，基于AI人设特征微调）
  - [x] 1.2: 修改 `selectParticipatingAIs` 中的阈值，将参与阈值从30提高到55，40~55区间50%概率发言，<40沉默
  - [x] 1.3: 修改 `calculateParticipationScore`，增加沉默概率权重，让AI更倾向于在非自己领域时选择沉默

- [x] Task 3: 重构AI评论生成逻辑
